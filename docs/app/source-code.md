# Source code

The **Source code** menu entry opens the MCP Ops Studio repository. The repository
contains the control plane, runtime worker, deployment builder, shared contracts,
database layer, sandbox executor, infrastructure, tests, and documentation.

Start with:

- [Platform development](../contributing/platform-development.md)
- [Architecture](../architecture.md)
- [Local development](../development.md)
- [Contribution guide](https://github.com/fabian-arnold/McpOpsStudio/blob/main/CONTRIBUTING.md)
- [Commit style](../commit-style.md)

The repository is the source of truth for product behavior and the documentation
published from `docs/`.
